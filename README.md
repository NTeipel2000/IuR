Zur besseren Ansicht mit klickbaren Dateipfaden im Browser:
- Venv starten und aktivieren
- Grip installieren: `pip install grip`
- Server starten: `grip <path to .md file` (`grip '.\src\abgabe1\Abgabe 1.md'`)
